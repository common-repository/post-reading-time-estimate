Add a fully customizable Medium-like estimated read time to your posts, pages, or custom post types.

== Description ==

Post Reading Time Estimate adds an estimated reading time to WordPress posts, pages, and custom post types; as seen on Medium. It also allows to choose what post types to display it on, change Words Per Minute to fit your audience read speed. It even comes with style settings including but not limited to Text Align, Font Weight, and Font Style!

You can access all plugin options on **Settings** &rarr; **Reading Time Options** screen in WordPress dashboard.

== Frequently Asked Questions ==

= Installation =

1. Upload the `post-reading-time-estimate` folder to the `wp-content/plugins` directory.
2. Activate it on `Plugins` screen in WordPress dashboard.

= Does it work with all themes? =

Sure thing! Post Reading Time Estimate creates very small footprint and is 100% compatible with your favorite theme.

= Can I add an estimate to custom post types? =

Yes! You can choose post types in **Settings** &rarr; **Reading Time Options**.

= Can I change Words Per Minute easily? =

It's actually super-easy! Head over to **Settings** &rarr; **Reading Time Options** and change **Words Per Minute (WPM)** setting to any number.
